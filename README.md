# Kabashi Altahir Portfolio Website

This is a simple academic portfolio website template designed for GitHub Pages.

## Files

- `index.html` — the main website content
- `style.css` — the website design
- `Kabashi_Altahir_CV.pdf` — replace this placeholder with your real CV PDF

## How to use

1. Create a GitHub account if you do not already have one.
2. Create a new repository called `Portfolio`.
3. Upload `index.html`, `style.css`, and your CV PDF.
4. Go to repository Settings.
5. Open Pages.
6. Under Branch, choose `main`.
7. Click Save.
8. Your website will become available at:

`https://YOUR-GITHUB-USERNAME.github.io/Portfolio/`

## What to edit

Replace:
- `your.email@ucdconnect.ie`
- `github.com/yourusername`
- `linkedin.com/in/yourprofile`
- Google Scholar link
- ORCID link
- publications
- awards
- CV file
